* Convert ```switch``` statement to ```if```: 
```
switch (browser) {
 case 'Edge':
 alert( "You've got the Edge!" );
 break;
 case 'Chrome':
 case 'Firefox':
 case 'Safari':
 case 'Opera':

 break;
 default:
 alert( 'We hope that this page looks ok!' );
}
```
